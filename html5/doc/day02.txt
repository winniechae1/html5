
a 태그 
다른 사이트 혹은 다른 문서로 이동할 때 사용하는 태그 

형식]
	<a href="이동할 사이트 혹은 문서">
	
참고]
	절대 경로: 시스템 혹은 사이트의 특정 문서의 위치를 말한다.
	상대 경로: 현재 보고 있는 문서를 기준으로 위치를 찾아가는 방식
	
	
입력 태그 
데이터를 입력받는 용도의 태그 

input : 단일행 입력 태그
inline 태그 
예]	이름, 아이디 ...

속성]
	type :  이 입력태그가 어떤 용도인지를 알려주는 속성
속성값]
	text, file, password, date, ...
	button, submit, reset, ...

	name
	id

textarea : 다중행 입력 태그 
block 태그 
예]	게시글

속성]
	name
	id
	cols, rows : 입력받을 수 있는 행과 열의 수를 지정하는 옵션
	
*****
입력태그는 반드시 name 속성값을 부여해주는 것이 원칙이다.
입력태그는 form태그로 감싸주는 것이 좋다.
